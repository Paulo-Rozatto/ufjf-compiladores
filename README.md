## Instalação
### Clonar o repositório
```bash
git clone https://github.com/Paulo-Rozatto/ufjf-compiladores.git
```

### Compilar
```bash
cd ufjf-compiladores/
mvn clean package
```

### Rodar
```bash
java -jar target/ufjf-compiladores-1.0.jar testes/sintaxe/exemplo1.txt
```
